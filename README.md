# Zirconium
Octave Species

---

### Download

    git clone https://github.com/reidiiius/zirconium.git

---

### Usage
command line

    cd zirconium/

    g++ -o paperboard cardstock.cpp

    ./paperboard

Selection:

        bfbf
        cgdae
        eadgbe
        fkbjdn
        ennead

View output:

        ./paperboard eadgbe | sensible-pager

Save output:

        ./paperboard eadgbe > exchequer-$(date +'%s').txt
